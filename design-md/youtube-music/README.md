# YouTube Music iOS Inspired Design System

Design system docs inspired by the [YouTube Music iOS app](https://apps.apple.com/us/app/youtube-music/id1017492454). Not the official system. Brand colors, font names, and component patterns are cross-referenced with YouTube Music's public product UI and Google's YouTube brand guidelines; exact private tokens may differ from production.

## Signature Moves

- **Near-black canvas** (`#030303`) — darker than Spotify's `#121212` so the immersive art backdrop glows
- **Immersive Now Playing** — a blurred, up-scaled copy of the current album art as a full-bleed backdrop
- **Song / Video toggle** — the signature pill under the art; flips audio ↔ official music video, no restart
- **White play button** — a `#FFFFFF` circle with a `#030303` glyph (NOT red — red is chrome, white is action)
- **YT Music Red** (`#FF0000`) is the ONLY accent — logomark, dragging scrubber, Premium/Subscribe CTAs, badges
- **Scrubber recolors on drag** — `#FFFFFF` at rest, `#FF0000` only while the user is actively scrubbing
- **YouTube-family greys** — `#AAAAAA` secondary, `#717171` tertiary, shared with the main YouTube app
- **Roboto everywhere** — the YouTube system face; never substituted for SF Pro or a serif
- **Up-next autoplay shelf** — a swipe-up queue generated by YouTube's recommendation engine
- **White-active tab bar** — Home / Samples / Explore / Library; no tint pill, no red on the active tab
- **Dark-only** — there is no light mode on iOS; the contrast against album art is the whole point
