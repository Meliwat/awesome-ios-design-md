# Turo — iOS design spec

Peer-to-peer car sharing. The vehicle photo is the product on a near-black canvas `#0F0F12`, with Turo Purple `#593BFB` as the single booking action, Turo Teal `#5CE0B8` as the trust/value accent (All-Star Host, instant-book), a teal-iconed specs strip, and a sticky price + Book bar

The complete **Turo** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
