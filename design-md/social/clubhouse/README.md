# Clubhouse — iOS design spec

live-audio social rooms. Warm cream "hallway" canvas (`#F2EFE4`) with DM Serif room titles, a grid of circular speaker avatars, and an emerald speaking-pulse ring (`#38B569` + `rgba(56,181,105,0.18)` halo) — all-pill controls, warm `#1A1A1A` dark mode, no blue anywhere.

The complete **Clubhouse** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
