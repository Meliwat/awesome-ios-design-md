# Lemon8 — iOS design spec

Lifestyle & social discovery. Bright editorial white canvas (`#FFFFFF`), two-column staggered masonry feed of magazine-cover cards, sparing Lemon8 Yellow (`#FFE600`) paired only with ink `#1A1A1A`, and a pastel topic-tag system (mint `#E4F5EC`, blush `#FCE8EE`, sky `#E6F0FB`).

The complete **Lemon8** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
