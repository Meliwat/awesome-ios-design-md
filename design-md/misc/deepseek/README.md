# DeepSeek — iOS design spec

AI reasoning chat assistant. A near-black canvas (#0E0E10) with exactly one accent — DeepSeek Blue (#4D6BFE) on the whale mark, user bubble, and active DeepThink/Search toggles; the signature is the reasoning trace: a recessed #16171A panel (darker than the canvas) with a 2pt blue left-bar holding the R1 chain-of-thought in dimmed italic #8A8B90, subordinate to an upright #ECECEC answer.

The complete **Deepseek** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
