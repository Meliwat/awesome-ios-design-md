# Postman — iOS design spec

API client and testing console. The request builder IS the product: a neutral-grey #1A1A1A canvas, the HTTP-method color system (GET #6BDD9A, POST #FFE47A, PUT #74AEF6, DELETE #F79090) read before text everywhere a method appears, a single Postman Orange #FF6C37 reserved for the Send button and active tab, and a syntax-colored JSON viewer that sinks onto a darker #161616 surface.

The complete **Postman** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
