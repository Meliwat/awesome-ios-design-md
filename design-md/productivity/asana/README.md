# Asana — iOS design spec

project & task management. The task list IS the product: 20pt completion circles that fill green (#62D26F) on tap, a single coral accent (#F06A6A) for the FAB and active tab, multicolor object pills as ~16% tints (plum #4573D2, aqua #4ECBC4), and a charcoal dark canvas (#1E1F21).

The complete **Asana** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
