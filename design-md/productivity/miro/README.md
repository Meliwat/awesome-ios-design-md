# Miro — iOS design spec

collaboration whiteboard. An infinite pinch-zoom canvas with a zoom-scaled dotted grid; Miro Yellow (#FFD02F) on Miro Ink (#050038) is the brand, while Miro Blue (#4262FF) is reserved for selection; tight 4pt-radius pastel sticky notes (default #FEF3B6) lift off the board with soft shadows, and a floating toolbar fills the active tool yellow.

The complete **Miro** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
