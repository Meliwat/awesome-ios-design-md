# Fitbit — iOS design spec

Supportive health dashboard. Fitbit Teal `#00B0B9` (brightening to `#21D9CE` on dark) on a teal-black `#001017` canvas, a per-metric color system (heart-rate coral `#FF6B81`, sleep purple `#7C5CFF`, readiness lime `#B8E986`), and the steps progress-ring hero with big-number-first DM Sans tiles.

The complete **Fitbit** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
