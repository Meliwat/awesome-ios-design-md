# Apple Fitness — iOS design spec

Activity tracking and Fitness+ streaming. True-black canvas `#000000` with the three Activity rings as the entire brand (Move `#FA114F`, Exercise `#92E82A`, Stand `#1EE4E1`) on 22%-opacity tracks, grouped system cards `#1C1C1E`, Apple’s label-opacity text ramp, a Move-pink chrome accent `#FF375F`, and a cinematic Fitness+ shelf with frosted-glass overlays.

The complete **Apple Fitness** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
