# Skype — iOS design spec

Microsoft's call-first messenger. Cloud-blue brand (bright #00AFF0 buttons/FABs, deep #0078D4 outgoing bubble), gray incoming (#EBEBEF / #2A2A30), inline call cards, a gradient video-call grid with circular controls, presence dots (green #2DC26B / away #FFC400), and a neutral near-black (#16161A) dark canvas.

The complete **Skype** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
