# Zalo — iOS design spec

Messaging super-app. Single Zalo Blue anchor (`#0068FF`) on every header, CTA, and active tab, white canvas over a calm blue-grey chat backdrop (`#E8ECF1`), asymmetric clipped-tail bubbles (incoming `#FFFFFF`, outgoing `#DBEBFF`), and a mini-app service launcher grid.

The complete **Zalo** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
