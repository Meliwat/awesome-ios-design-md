# Deezer — iOS design spec

Music streaming, gradient dark. Violet near-black #0F0D13 canvas with a scarce purple-to-pink gradient (#A238FF to #FF0092) lighting only the 68pt circular play button, the scrubber fill, and the Flow badge; the signature is the living-gradient Flow artwork with an embedded white equalizer and pink now-playing rows.

The complete **Deezer** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
