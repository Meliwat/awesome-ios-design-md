# ESPN — iOS design spec

fast live-first sports scoreboard platform. Near-black canvas (#0E0F11, never pure black) with a single ESPN Red accent (#D50A0A), a pinned horizontal scores ticker, pulsing #FF1A1A LIVE badges, and the winning team's score in green (#1FAA59) with tabular numerals.

The complete **Espn** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
