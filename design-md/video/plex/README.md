# Plex — iOS design spec

calm personal-media server platform. Cool charcoal canvas (#1F2326, never true black) with a single Plex Yellow accent (#E5A00D) and dark #1A1304 button text, an On Deck resume rail with yellow progress bars, and a multi-server status picker.

The complete **Plex** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
