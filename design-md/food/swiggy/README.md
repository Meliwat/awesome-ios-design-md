# Swiggy — iOS design spec

appetite-driving food delivery and quick-commerce platform. A single Swiggy Orange accent (#FC8019) over a near-black canvas (#121212), photo-forward restaurant cards with a #000-fading bottom scrim and bold offer text, a green rating chip (#48C479), and a floating ADD button that morphs into a stepper.

The complete **Swiggy** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
