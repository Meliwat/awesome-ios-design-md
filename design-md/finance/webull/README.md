# Webull — iOS design spec

Dark-native retail trading terminal. A near-black canvas `#0B0E11` with the Webull blue→cyan gradient `#1B9EFB → #20D5C4`, a full-bleed candlestick chart, a depth-bar order-book ladder, an options chain, and pervasive saturated up `#00C076` / down `#FA5252` color on a split Buy/Sell docked pair.

The complete **Webull** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
