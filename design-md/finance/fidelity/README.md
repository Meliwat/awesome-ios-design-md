# Fidelity — iOS design spec

Institutional brokerage and portfolio tracking. Fidelity Green `#368727` + Heritage Green `#00754A` on a calm evergreen dark canvas `#0E1411`, a 34pt tabular account-value hero with sparkline and range tabs, flat hairline holding rows, and sacred gain `#15B374` / loss `#E5544B` numerals with a sticky green Trade bar.

The complete **Fidelity** design system — free. The framework-neutral `DESIGN.md` plus SwiftUI, Expo, and Jetpack Compose companions: screen-by-screen breakdown, full color & type system, component specs, motion curves, haptics, and a ready-to-paste AI-agent implementation prompt.
