# DoorDash iOS Inspired Design System

[DESIGN.md](DESIGN.md) extracted from observation of the public [DoorDash iOS app](https://apps.apple.com/us/app/doordash-food-delivery/id719972451). This is not the official design system. Colors, fonts, and spacing may not be 100% accurate — but it's a solid starting point for building e-commerce / delivery UI that feels like DoorDash.

## Signature Moves

- **DoorDash Red** (`#EB1700`) used aggressively for every "spend money" action
- **Photo-first merchant cards** with 16:9 food photography, 16pt radius
- **Floating checkout CTA** — pill-rounded red button pinned bottom when cart > 0
- **Semantic accents**: Rating orange (`#FF8000`), $0 Delivery green (`#008B4A`)
- **TT Norms Pro** bold titles (weight 700) — assertive, not elegant
- **Horizontal carousels by intent**: "Fastest near you", "Order it again", "$0 delivery fee"
